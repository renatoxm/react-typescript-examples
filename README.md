## Install the project

This project runs on [Vitejs](https://vitejs.dev/guide/)

```sh
# pnpm
pnpm i
```

## Running

To start the project locally, run:

```sh
pnpm dev
```

Open `http://localhost:5173` with your browser to see the result.
